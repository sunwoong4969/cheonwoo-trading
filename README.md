# 천우무역 홈페이지

천우무역 회사의 공식 홈페이지입니다.

## 프로젝트 개요

글로벌 무역 회사인 천우무역의 공식 웹사이트로, 회사 소개, 서비스, 제품, 연락처 정보를 제공합니다.

## 기술 스택

- **HTML5**: 웹 페이지 구조
- **CSS3**: 스타일링 및 반응형 디자인
- **JavaScript (ES6+)**: 인터랙티브 기능
- **Google Fonts**: Noto Sans KR 폰트

## 파일 구조

```
.
├── index.html      # 메인 HTML 파일
├── style.css       # 스타일시트
├── script.js        # JavaScript 로직
└── README.md       # 프로젝트 문서
```

## 주요 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기 지원
- **스무스 스크롤**: 부드러운 페이지 네비게이션
- **애니메이션**: 스크롤 시 요소 등장 애니메이션
- **문의 폼**: 연락처 문의 폼
- **모바일 메뉴**: 모바일 환경을 위한 햄버거 메뉴

## 섹션 구성

1. **헤더**: 고정 네비게이션 바
2. **히어로 섹션**: 메인 비주얼 및 CTA
3. **회사소개**: 회사 개요 및 통계
4. **서비스**: 주요 서비스 소개
5. **제품**: 거래 품목 카테고리
6. **연락처**: 연락처 정보 및 문의 폼
7. **푸터**: 추가 정보 및 링크

## 실행 방법

### 방법 1: 직접 파일 열기
`index.html` 파일을 브라우저에서 직접 열기

### 방법 2: 로컬 서버 사용 (권장)

**Python 사용:**
```bash
python -m http.server 8000
```

**Node.js 사용:**
```bash
npx http-server -p 8000
```

그 다음 브라우저에서 `http://localhost:8000` 접속

## 커스터마이징

### 색상 변경
`style.css` 파일의 `:root` 변수를 수정하여 색상을 변경할 수 있습니다:

```css
:root {
    --primary-color: #2563eb;    /* 주요 색상 */
    --secondary-color: #1e40af;  /* 보조 색상 */
    --accent-color: #3b82f6;      /* 강조 색상 */
}
```

### 내용 수정
`index.html` 파일에서 텍스트 내용을 직접 수정할 수 있습니다.
